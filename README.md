# domain-purpose-classifier-organizer

Chrome の保存済みブックマークを、ドメイン、サイト目的、用途、テーマなどの複数軸で分類し、フォルダ整理の差分プレビュー、適用、直前復元まで扱う Chrome Extension です。

## MVP

- 保存済みブックマークツリーを読み込む。
- ローカルのルールベースで分類候補と分類理由を出す。
- Side Panel で分類結果と移動予定先を確認する。
- 手動テスト前に現在のブックマーク配置をバックアップする。
- 確認後にフォルダ作成とブックマーク移動を実行する。
- 直前ジョブの復元情報を保存し、可能な範囲で元の親フォルダへ戻す。
- 保存済みバックアップから、テスト前の親フォルダと順序へ戻す。
- 整理時に HTTP/HTTPS ブックマークのリンク切れを確認し、HTTP 404/410 の候補だけを確認後に削除する。
- Options で分類先設定、サイト割り当て、候補生成、ルール管理、JSON 直接編集の各モードで編集する。
- Side Panel 内に Options を埋め込み表示でき、必要に応じて従来どおり別タブの Options も開ける。
- 現在のブックマークをホスト名とドメイン名でサイト一覧化し、分類先へドラッグ & ドロップしてドメインルールを作成または更新する。
- サイト割り当てでは分類済みサイトを非表示にでき、サイトカードには favicon サムネイルを表示し、長いサイト名やサンプル名はカード内で省略表示する。
- 現在のブックマークだけでなく、主要検索、ポータル、業務、開発、ショッピング、ニュースなどの代表的なサイトからも候補ルールを作成する。
- 候補ルール作成時に、目的、用途、テーマの対象軸を選択する。
- 分類先は大項目と中項目で管理し、整理時は `分類済みブックマーク / 大項目 / 中項目` の階層へ配置する。
- 分類先ごとに組み込みアイコンまたは画像 URL のサムネイルを設定できる。
- 初期ルールにはゲーム、成人向け、削除予定の分類先を含める。
- `整理 / 削除予定` に分類されたブックマークは、整理適用時の確認後に削除する。
- 整理適用後に空になったフォルダを削除する。

## 開発

```powershell
npm test
```

Chrome で手動確認する場合は、`chrome://extensions` の「パッケージ化されていない拡張機能を読み込む」から次のフォルダを指定します。

```text
D:\AI\ChromeExtension\domain-purpose-classifier-organizer\extension
```

## 権限

- `bookmarks`: ブックマークツリー取得、フォルダ作成、ブックマーク移動に使う。
- `storage`: 分類ルール、実行履歴、直前復元用スナップショット、テスト用バックアップの保存に使う。
- `sidePanel`: 一覧確認、差分プレビュー、実行導線を Side Panel に表示する。
- `http://*/*`、`https://*/*`: リンク切れチェック時に HTTP 状態を確認するために使う。HTTP 404/410 のみ削除候補とし、通信エラー、認証エラー、一時的なサーバーエラーは削除候補にしない。

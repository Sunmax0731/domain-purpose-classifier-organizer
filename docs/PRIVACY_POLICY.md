# Bookmark Purpose Organizer プライバシーポリシー

最終更新日: 2026-05-07

Bookmark Purpose Organizer は、Chrome に保存されたブックマークを分類ルールに基づいて整理するための Chrome 拡張機能です。本ポリシーでは、この拡張機能が扱うデータ、利用目的、外部送信、共有、保存について説明します。

## 収集または利用するデータ

この拡張機能は、機能提供のために次のデータを端末上で読み取ります。

- Chrome に保存されているブックマークのタイトル、URL、フォルダ構成、並び順
- ユーザーが作成または編集した分類ルール、分類先、アイコン設定、優先度
- 整理前後のプレビュー、直前復元用スナップショット、テスト用バックアップ
- リンク切れチェックの結果

これらのデータは、ブックマークの分類、整理前プレビュー、バックアップ、復元、リンク切れ候補の確認のためだけに使用します。

## 外部送信

この拡張機能は、分類、候補生成、整理、復元のためにブックマークの完全な URL、タイトル、フォルダ構成を外部の分類 API、LLM、広告サービス、分析サービスへ送信しません。

ただし、次のユーザー向け機能ではネットワーク通信が発生します。

- リンク切れチェック: ユーザーが保存している HTTP/HTTPS ブックマーク URL にリクエストし、HTTP 404/410 などの状態を確認します。
- サイトのアイコン表示: サイト割り当て画面で favicon を表示するため、ドメイン名を外部 favicon サービスへ送信する場合があります。

ページ本文の読み取り、フォーム内容の収集、任意サイトへのコード注入、広告目的のトラッキングは行いません。

## 保存場所

分類ルール、バックアップ、復元用スナップショット、リンク切れチェック結果は、Chrome の `chrome.storage.local` を使ってユーザーのブラウザ内に保存します。開発者のサーバーには保存しません。

## 第三者提供

この拡張機能は、ユーザーデータを販売、貸与、広告目的で共有、またはデータブローカーへ提供しません。ユーザーデータを第三者へ転送することもありません。ただし、上記のリンク切れチェックおよび favicon 表示のために、ユーザー操作に基づくネットワーク通信が発生する場合があります。

## リモートコード

この拡張機能は、外部サーバーから JavaScript などのリモートコードを読み込んで実行しません。拡張機能の動作に必要なコードは、配布パッケージ内に含まれています。

## Chrome Web Store User Data Policy への準拠

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

## ユーザーによる管理

ユーザーは、Chrome の拡張機能管理画面からいつでもこの拡張機能を無効化または削除できます。拡張機能内で作成した分類ルールやバックアップは、Chrome の拡張機能データ削除操作により削除できます。

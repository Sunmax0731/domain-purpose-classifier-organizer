# 05 テスト計画: ブックマーク分類・フォルダ自動整理

## Functional Tests

- 既存ブックマークツリーを読み込める。
- ドメイン分類が期待どおりに候補を出す。
- 用途ルールとテーマルールを追加できる。
- 差分プレビューが移動先フォルダを正しく表示する。
- 実行後にブックマークフォルダ構造が期待どおりになる。

## Failure and Recovery Tests

- 権限不足や API エラー時に失敗内容を表示する。
- 無効ルールが保存されない。
- 実行途中でキャンセルしても整合性が崩れない。
- ロールバックで元の構造へ戻せる。

## Manual Test Checklist

- 100 件以上のブックマークで一覧の操作性を確認する。
- 同一ドメインで用途が異なるページを分類補正できる。
- 既存フォルダとのマージ方針が分かりやすい。
- 危険操作前の確認 UI が十分に明確である。

## Chrome Web Store 公開前テスト

- 新規インストール、更新、無効化、再有効化、削除を確認する。
- `permissions` と `host_permissions` の要求が説明どおりであることを確認する。
- `service worker` の再起動後も主要機能が維持されることを確認する。
- ストア掲載用スクリーンショットと実 UI が一致していることを確認する。
- 単一目的の説明、`Privacy Policy`、サポート導線が利用者に分かることを確認する。

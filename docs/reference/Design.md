# デザインガイド: ChromeExtension

## 目的

この拡張機能の UI は、ブックマークを安全に再整理するための判断を助けることを目的とする。

## UI 方針

- 小さな Popup に機能を詰め込まず、主要作業は `Side Panel` で扱う。
- 分類候補と分類理由を隣接表示し、修正しやすくする。
- 実行前後の差分がすぐ比較できる配置にする。
- 危険操作は色、文言、確認ステップで明確に区別する。

## 画面の役割

- Popup
  - 直近状態の要約
  - Side Panel を開く導線
- Side Panel
  - ブックマーク一覧
  - 候補分類
  - 差分プレビュー
  - 実行確認
- Options
  - 分類ルール
  - 除外条件
  - バックアップ設定

## UX 上の注意

- 一括実行前に、影響件数と作成フォルダ数を明示する。
- 自動分類を過信させず、手動補正を前提にした文言にする。
- 既存フォルダと新規フォルダを視覚的に区別する。

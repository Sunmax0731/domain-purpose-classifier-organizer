# Development Workflow

## 推奨の作業順

1. ブックマーク取得の最小実装を作る。
2. ルールベース分類を実装する。
3. 分類理由と差分プレビューを表示する。
4. 実行処理とロールバックを追加する。
5. Options でルール編集を可能にする。
6. Chrome Web Store 出品前チェックを行う。

## 実装単位

- データ取得
- 分類ルール
- UI 表示
- 実行と復元
- ストア公開準備

## 完了条件

- ブックマーク一覧が取得できる。
- 複数軸分類が動く。
- フォルダ変更前にプレビューが見える。
- 変更後に復元できる。
- 権限説明とストア掲載文脈が揃っている。

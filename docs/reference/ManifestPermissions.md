# Manifest / Permissions

## 想定権限

- `bookmarks`
  - ブックマークツリー取得
  - フォルダ作成
  - ブックマーク移動
- `storage`
  - 分類ルール保存
  - 実行履歴保存
  - ロールバック情報保存
- `sidePanel`
  - 一覧整理 UI の表示

## 追加検討

- `tabs`
  - 現在ページを暫定分類対象へ追加する場合のみ検討する
- `contextMenus`
  - ページやリンクの右クリック追加を扱う場合のみ検討する

## 審査向け注意

- なぜ `bookmarks` 権限が必要かを明記する。
- 単一目的は「ブックマーク分類とフォルダ整理」で一貫させる。
- 外部通信を使う場合は、用途と送信内容を明文化する。

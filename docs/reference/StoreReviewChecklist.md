# Store Review Checklist

## Chrome Web Store 向け確認

- ストア説明文が実機能と一致している。
- 単一目的が「ブックマーク分類・フォルダ整理」で明確である。
- `bookmarks` 権限の理由が説明されている。
- `Manifest V3` 構成が整理されている。
- プライバシーポリシーとサポート導線がある。

## 機能確認

- ブックマーク取得
- 分類候補表示
- 差分プレビュー
- 実行確認
- ロールバック

## リスク確認

- 一括移動を誤実行しない導線になっている。
- 既存フォルダ破壊や重複作成の扱いが明確である。
- 権限過剰や説明不足が残っていない。

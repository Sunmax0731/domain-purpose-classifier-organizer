# ChromeExtension Idea Summary

## 対象アイデア

- No.12
- 名前: ブックマーク分類・フォルダ自動整理
- リポジトリ名: `domain-purpose-classifier-organizer`

## 概要

保存済みブックマークを、ドメイン、サイト目的、用途、業務カテゴリ、学習テーマなどの複数軸で分類し、分類結果に応じてブックマークフォルダへ整理する ChromeExtension。

## コア価値

- ブックマークそのものを整理対象にする
- ドメインだけでなく複数観点で分類する
- 分類結果をそのままフォルダ構造へ反映する
- ユーザーがルールを編集しながら再整理できる

# Chrome Web Store 入力項目

この文書は、Chrome Web Store Developer Dashboard の入力欄にそのまま貼り付けられる文面をまとめたものです。実装は `extension/manifest.json` の `bookmarks`、`storage`、`sidePanel`、`http://*/*`、`https://*/*` に合わせています。

## プライバシー

### 単一用途の説明

```text
保存済みブックマークを目的・用途・ドメインなどの分類ルールで整理し、適用前プレビュー、バックアップ、復元、リンク切れ確認を通じて安全にフォルダへ分類すること。
```

### bookmarks が必要な理由

```text
保存済みブックマークの一覧取得、分類候補の作成、適用前プレビュー、ユーザー確認後のフォルダ作成・移動・削除予定ブックマークの削除、復元用バックアップの作成に使用します。ブックマーク情報は分類・整理機能のためだけに扱い、外部分類APIへ送信しません。
```

### storage が必要な理由

```text
分類ルール、分類候補、整理プレビュー、直前復元用スナップショット、手動テスト用バックアップ、リンク切れチェック結果を端末内の chrome.storage.local に保存するために使用します。保存したデータはユーザーのブラウザ内でのみ利用します。
```

### sidePanel が必要な理由

```text
分類結果、変更プレビュー、バックアップ、リンク切れチェック、整理実行ボタンを Chrome の Side Panel に表示し、ブックマークを確認しながら操作できるようにするために使用します。
```

### ホスト権限が必要な理由

```text
リンク切れチェック時に、ユーザーが保存している HTTP/HTTPS ブックマーク URL へリクエストし、HTTP 404/410 などの削除候補を判定するために使用します。また、サイト割り当て画面の favicon 表示ではドメイン名を favicon サービスに渡す場合があります。ページ本文の読み取りや任意サイトへのコード注入は行いません。
```

### リモートコード

選択: `いいえ、リモートコードを使用していません`

理由欄が表示される場合:

```text
外部サーバーから JavaScript などのリモートコードを読み込んで実行しません。拡張機能の動作に必要なコードは配布パッケージ内に含まれています。リンク切れチェックの HTTP リクエストと favicon 画像表示はネットワーク通信であり、リモートコードの実行ではありません。
```

### データ使用

チェックするデータ種別:

- `ウェブ履歴`

理由: ブックマークの URL、タイトル、フォルダ構成を読み取り、リンク切れチェックでは保存済み URL にアクセスするためです。個人識別情報、健康、財務、認証、個人的コミュニケーション、位置情報、ユーザーのアクティビティ、ウェブサイトのコンテンツは収集しません。

チェックする宣誓項目:

- `承認されている以外の用途で第三者にユーザーデータを販売、転送しません`
- `アイテムの唯一の目的と関係のない目的でユーザーデータを使用または転送しません`
- `信用力を判断する目的または融資目的でユーザーデータを使用または転送しません`

## ストア掲載情報

### 商品名

```text
Bookmark Purpose Organizer
```

### 短い説明

```text
ブックマークを目的・用途・ドメインで分類し、プレビューとバックアップ付きで安全にフォルダ整理できます。
```

### 詳細な説明

```text
Bookmark Purpose Organizer は、増え続ける Chrome ブックマークを目的・用途・ドメインなどのルールで分類し、適用前に確認してから安全にフォルダ整理するための拡張機能です。

主な機能:
- 保存済みブックマークをスキャンし、分類候補と移動先を表示
- 大項目と中項目によるフォルダ構成を GUI で設定
- サイト割り当て画面で、現在のブックマークから抽出したサイトを分類先へドラッグ & ドロップ
- ルール一覧、グリッド表示、JSON 直接編集に対応
- 整理前に変更プレビューと確認ダイアログを表示
- テスト用バックアップ、直前復元、バックアップからの復元に対応
- HTTP 404/410 などのリンク切れ候補を確認して削除
- 空になった整理対象フォルダを自動削除

安全性:
- ブックマークの分類と整理は、ユーザー確認後に実行します。
- 分類、候補生成、整理、復元のためにブックマークの完全な URL やタイトルを外部分類 API へ送信しません。
- リンク切れチェックでは、ユーザーが保存している URL へ HTTP/HTTPS リクエストを行い、削除候補を確認します。
- favicon 表示では、ドメイン名を外部 favicon サービスへ送信する場合があります。
- 外部サーバーからリモートコードを読み込んで実行しません。

必要な権限:
- bookmarks: ブックマークの読み取り、フォルダ作成、移動、確認後の削除、復元に使用します。
- storage: 分類ルール、バックアップ、復元用スナップショット、リンク切れチェック結果を保存します。
- sidePanel: 分類結果、変更プレビュー、バックアップ、リンク切れチェックを Side Panel に表示します。
- http://*/* / https://*/*: リンク切れチェックのため、保存済みブックマーク URL の HTTP 状態を確認します。
```
